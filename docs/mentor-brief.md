# Mentor Brief (ТЗ)

Суть: спроектировать и реализовать упрощённый аналог Яндекс.Диск с возможностью масштабирования и обработки больших файлов.

## Задачи

1) Спроектировать систему хранения файлов, чтобы были учтены все требования.
2) Реализовать систему и подготовить к запуску (все подготовительные действия для первого релиза).

## Требования

- Пользователь должен иметь возможность загрузить файл в хранилище, скачать файл, получить список всех файлов, удалить файл.
- Можно загружать файлы до **1 ГБ**.
- Пользователей может быть несколько, пока без явной авторизации.

## Процесс

- Идём поступательно: каждый следующий шаг усложняет систему.
- Задачи будут как на реальной работе: не супер точные и достаточно абстрактные, поэтому сбор/досбор требований по задаче — тоже на тебе.
- Если проект устраивает — можно начинать делать и задавать вопросы.
